Introduction

This document serves as a comprehensive guide to web development, covering essential topics from HTMl to web publishing and ngrok.

HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a series of elements.

CSS

CSS (Cascading Style Sheet) is used to style the visual presentation of a document written in HTML. It defines how elements should be displayed on the screen.

JavaScript

JavaScript is a programming language that enables interactive web pages. It is commonly used for client-side scripting to make dynamic content and interactive elements.

Linux

Linux is an open-source operating system that is widely used in web development. It provides a powerful and flexible environment developing and hosting web applications

GitHub

GitHub is a platform for hosting and sharing code. It provides version control features for developers working on projects.

Git Commands

Git is a version control system used to track changes in files and coordinate work on those files among multiple people. Here are some common Git command:

  • git init
  • git add
  • git push
  • git pull
  • git commit
Web Publishing

Web publishing involves making your website availabe to the public on the internet. It includes tasks such as uploading files to a web server and configuring domain settings.

Ngrok

Ngrok is a tool creates secure tunnels to localhosr. It allows you to expose a local server on the internet for testing and development purposes.
